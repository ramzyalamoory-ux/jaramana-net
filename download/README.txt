## 🚀 طريقة التشغيل:

### المتطلبات:
- Node.js 18+ أو Bun
- npm أو bun

### الخطوات:

1️⃣ **فك ضغط الملف**
```bash
unzip jaramana-net.zip
cd jaramana-net
```

2️⃣ **تثبيت المكتبات**
```bash
bun install
# أو
npm install
```

3️⃣ **إنشاء قاعدة البيانات**
```bash
npx prisma db push
```

4️⃣ **تشغيل الموقع**
```bash
bun run dev
# أو
npm run dev
```

5️⃣ **فتح المتصفح**
```
http://localhost:3000
```
